# Тестовое задание "Вектора"
## Program_1
Программа генерирует  N рандомных векторов случайной размерности m и записывает их в файл vectors.csv.

## Program_2
Программа считывает векторы из файла vectors.csv и вычисляет расстояния между парами векторов.
Затем строится гистограмма распределения этих расстояний.

### Для запуска проекта достаточно запустить  Program_2

### Алгоритм построения гистограммы
1. Создание пустого списка histogram, который будет содержать количество расстояний в каждом из интервалов.
2. Для каждого расстояния в списке distances вычисляется индекс соответствующего интервала в списке histogram 
с помощью операции целочисленного деления (//) и остатка от деления (%).
3. Увеличивается значение соответствующего элемента списка histogram на 1.
4. Гистограмма строится с помощью функции plt.hist из библиотеки matplotlib.
5. Задаются подписи осей и заголовок гистограммы с помощью функций plt.xlabel, plt.ylabel и plt.title.
6. Гистограмма отображается на экране с помощью функции plt.show.

### Переменные
- N: количество векторов
- m: размерность векторов
- step: шаг интервала в гистограмме
- vectors: список векторов, считанных из файла
- distances: список расстояний между парами векторов
- min_distance: минимальное расстояние и номер пары векторов, между которыми оно достигается
- max_distance: максимальное расстояние и номер пары векторов, между которыми оно достигается
- histogram: список количества расстояний в каждом из интервалов гистограммы
